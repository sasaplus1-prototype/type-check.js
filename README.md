# type-check.js

type check

## Installation

```sh
$ npm install sasaplus1-prototype/type-check.js
```

## Usage

via `require()`

```js
var typeCheck = require('type-check');
```

via `<script>`

```html
<script src="type-check.min.js"></script>
```
